Django Model

Python

Implementación de modelo Django con relaciones y métodos personalizados para el proyecto Yosise. 

from django.db import models

class Usuario(models.Model):
    nombre = models.CharField(max_length=100)
    email = models.EmailField(unique=True)
    fecha_registro = models.DateTimeField(
        auto_now_add=True
    )
    
    def __str__(self):
        return self.nombre

Class-Based View

Python

Vista basada en clases para listar y filtrar registros con paginación. 

from django.views.generic import ListView

class UsuarioListView(ListView):
    model = Usuario
    template_name = 'usuarios.html'
    context_object_name = 'usuarios'
    paginate_by = 10
    
    def get_queryset(self):
        return Usuario.objects.filter(
            activo=True
        ).order_by('-fecha_registro')

SQL Query Optimizada

SQL

Consulta SQL con JOIN y agregación para análisis de datos. 

SELECT u.nombre, 
       COUNT(p.id) as total_posts
FROM usuarios u
LEFT JOIN posts p ON u.id = p.usuario_id
WHERE u.activo = 1
GROUP BY u.id, u.nombre
HAVING COUNT(p.id) > 5
ORDER BY total_posts DESC;

Django Form

Python

Formulario con validación personalizada y limpieza de datos. 

from django import forms

class RegistroForm(forms.ModelForm):
    class Meta:
        model = Usuario
        fields = ['nombre', 'email']
    
    def clean_email(self):
        email = self.cleaned_data['email']
        if Usuario.objects.filter(
            email=email
        ).exists():
            raise forms.ValidationError(
                'Email ya registrado'
            )
        return email

Fetch API

JavaScript

Petición asíncrona para consumir API REST con manejo de errores. 

async function obtenerDatos() {
    try {
        const response = await fetch('/api/usuarios/');
        
        if (!response.ok) {
            throw new Error('Error en la petición');
        }
        
        const data = await response.json();
        return data;
    } catch (error) {
        console.error('Error:', error);
    }
}
